##  Features

###  1. Task Management  
- Add tasks  
- Prevent duplicates  
- List all saved tasks  
- Stored in memory + future DB support  

###  2. AI-Powered Agent  
Powered by **Gemini 1.5 Flash**, supports:  
- AI Day Planner  
- Healthy Indian Meal Suggestions  
- Weekly Household Shopping List  
- Natural language reasoning & generation  

###  3. FastAPI Backend  
Clean API endpoints:  
- `/add_task`  
- `/list_tasks`  
- `/ai_plan_day`  
- `/ai_meal_suggestions`  
- `/ai_shopping_list`

###  4. UI Ready  
A simple HTML/JS UI is provided to interact with the agent.
